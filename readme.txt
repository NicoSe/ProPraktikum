2. Problembehebung
----------------------------------------------------------------------------------------

Falls du Probleme mit der Bildschirmanzeige hast oder Schwierigkeiten beim Starten des
Spiels auftreten, oder wenn das Spiel abstürzt und zum Desktop zurückkehrt, kannst du
entweder die Einstellungen des Spiels oder die Systemkonfiguration deines Computers
ändern. Bitte stelle in einem solchen Fall zunächst sicher, dass du die aktuellsten
Treiber für deine Hardware verwendest.

--------------------------------------------------------------------------------
2.1. Installation der Grafik- und Mainboardtreiber

Wenn Sie Grafik-Probleme feststellen sollten, dann installieren Sie bitte die
aktuellsten verfügbaren Treiber für Ihre Grafikkarte und das Mainboard.
Links zu den bekanntesten Herstellern von Grafikkarten und Mainboards
finden Sie hier:

NVIDIA (GeForce)				            ATI (Radeon)
http://www.nvidia.de (Deutsch)			    http://www.ati.de (Deutsch)
http://www.nvidia.com (International)		http://www.ati.com (International)

--------------------------------------------------------------------------------
2.2. Installation der Soundtreiber

Wenn Sie Sound-Probleme feststellen sollten, dann installieren Sie bitte die
aktuellsten verfügbaren Treiber für Ihre Soundkarte.

--------------------------------------------------------------------------------
2.3. Bugs und Kommentare

* Es ist _nicht_ ratsam, das Spiel in das Windows-Verzeichnis (normalerweise
  C:\Windows), ein Unterverzeichnis des Windows-Verzeichnisses, oder das
  Stammverzeichnis des Systemlaufwerks (normalerweise C:\) zu installieren.

* Es wird empfohlen, alle anderen laufenden Programme zu beenden, bevor Sie das
  Spiel installieren, um die Gefahr für Softwarekonflikte zu verringern.

* Wenn Sie Realtek High Definition Audio, Soundmax HD Audio oder nForce Audio
  benutzen, könnte es zu Anomalien bei der Soundwiedergabe kommen. Um dieses
  Problem zu umgehen, stellen Sie den Schieberegler für die Hardwarebeschleunigung
  in der Windows-Systemsteuerung -> Sounds und Audiogeräte -> Audio -> Erweitert
  -> Systemleistung auf 'Basic'.

* Einige SFX-Effekte werden zu laut abgespielt, wenn die Lautstärke für SFX auf 100
  eingestellt ist.

3. Systemanforderungen
---------------------------------------------------------------------------------------

EMPFOHLENE KONFIGURATION
Windows 7/8/10 PC (32 oder 64 Bit)
Prozessor: Octa-Core-Prozessor von Intel oder AMD
1 TB RAM
10 TB freier Festplattenspeicher
DirectX 9.0c-kompatible NVIDIA oder AMD ATI-Grafikkarte mit 6 GB RAM
(NVIDIA Ge Force RTX 2080ti oder besser; ATI Radeon 5900XT oder besser).
DirectX-kompatible Soundkarte
Glasfaser Internetzugang
Java Version 1.8.0 oder höher

MINIMALE KONFIGURATION
Windows 7/8/10 PC (32 oder 64 Bit)
Prozessor: Intel Dual-Core 2,0 GHz-Prozessor oder gleichwertig
2 GB RAM
30 MB freier Festplattenspeicher
DirectX 9.0c-kompatible Grafikkarte mit 512 MB RAM
DirectX-kompatible Soundkarte
Internetzugang
Java Version 1.8.0 oder höher

5. Installationsanleitung
---------------------------------------------------------------------------------------

    1. Führen Sie die Battleships.exe Datei aus!
    2. Folgen Sie den Anweisungen der Installation.
       Um das SPiel später leichter starten zu können machen Sie ein Hacken bei der
       Option "Shortcut am Desktop erstellen".
    3. Im Installationsverzeichnis oder wenn die Sie den Tipp von 2. befolgt haben
       am Desktop, finden sie eine Datei Battleships.exe.
    4. Zum Spielen führen Sie die Datei "Battleships.exe" aus!

    Hinweis : Informationen zum Compilieren einer .jar Datei lesen Sie sich die
              compile.txt durch.
